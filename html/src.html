class src.atom_class.atom(levels, state='none')[source]¶

this class stores information for an atom.

Variables:
  • energy_levels – a list of increasing energy levels
  • state – the density matrix of the initial internal state of the atom
  • N – the dimension of the atom-space
  • trans – a list of pairs for all transitions possible. each pair contains [transition frequency, transition matrix], where the transition matrix is 0 except for one entry that is 1. There are N * (N - 1) / 2 transitions.
