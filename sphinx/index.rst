Welcome to Trapped Ion Sim's documentation!
===========================================

How to get it:

- Follow the installation guide on: http://qutip.org/docs/3.0.0/installation.html and install qutip
- Download the trapped_ion_sim repo (https://github.com/QMSimProject/trapped_ion_sim) or clone it with git

Python 2.7:

- https://docs.python.org/2/

QuTip 3.0:

- http://qutip.org/
- http://qutip.org/docs/3.0.0/index.html

Start by looking at the qutip_example folder, then experiment with the GUI (test_gui.py) and take a look at test_integration.py to see how it is done without the GUI. All functions and classes can be found here: :ref:`modindex`
